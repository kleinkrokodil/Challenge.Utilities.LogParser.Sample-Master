# Challenge.Utilities.LogParser.Sample-Master
This is a sample application making use of Challenge.Utilities.LogParser

## Getting Started

Using GitHub, select clone to clone the project to your repository

Using Visual Studio, switch views to the solution in solution explorer

Using Visual Studio, click Build > Build Solution from your menu 

### Prerequisites

Visual Studio 2019 16.7 and higher

.NET Core 3.1

### Installation

Extract/Clone and build Challenge.Utilities.LogParser.Sample-Master
